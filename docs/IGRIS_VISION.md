# IGRIS Vision

IGRIS (Intelligent Git Repository Interaction Service) aims to be an open, transparent and locally deployable alternative to commercial autonomous coding agents.  The goals of the project are to provide:

- **Controllability** – users maintain oversight of all actions and can disable or approve features as needed.
- **Local deployment** – the default configuration runs entirely on an Ubuntu server using a local Ollama model.
- **Cost awareness** – the agent chooses between local, remote or cloud LLMs based on cost and capability.
- **Safety** – a built‑in anti‑loop mechanism and safety rules prevent harmful or runaway actions.
- **Extendability** – a modular architecture allows adding new tools, models or advisory layers.

The long term vision is to build a fully autonomous engineering console capable of understanding a repository, planning modifications, executing tests and generating pull requests while always providing the operator clear insight and control.
